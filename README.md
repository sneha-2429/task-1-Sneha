# Personal Portfolio Website

## Overview
A responsive personal portfolio website developed using HTML, CSS, and JavaScript. The website presents personal information, skills, projects, and contact details in a clean and modern interface.

## Features
- Responsive navigation menu
- Hero section with introduction
- About Me section
- Skills showcase
- Project portfolio section
- Contact form validation
- Mobile-friendly design
- Smooth scrolling navigation

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
Portfolio/
│
├── index.html
├── style.css
├── script.js
└── README.md

## Installation
1. Clone the repository:
   git clone <repository-url>

2. Open the project folder.

3. Run `index.html` in your browser.
